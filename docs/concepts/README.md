# Концепты

Идеи, которые встречаются в нескольких книгах программы. Каждый концепт — отдельная заметка, на которую ссылаются разные книги и режимы. Граф Obsidian (Ctrl+G) покажет пересечения — это подсказка для cross-program фич.

## Запланированные концепты

- [[boundaries|Границы]] — Glover (центральный навык) + Chapman (в отношениях) + Berne (вместо игр)
- [[self-soothing|Self-soothing]] — Glover (эмоциональная саморегуляция)
- [[communication|Коммуникация чувств]] — все три книги, с разных ракурсов
- [[shame|Стыд]] — Glover, частично Berne
- [[needs|Потребности]] — Glover (свои потребности) + Chapman (языки любви как потребности)

## Структура заметки о концепте

```markdown
# Название концепта

## Определение
Одним абзацем — что это такое вообще.

## В книгах программы

### У Glover
[[nice-guy-glover#раздел]] — как подаётся у автора.

### У Berne
[[games-people-play-berne#раздел]] — как у автора.

### У Chapman
[[love-languages-chapman#раздел]] — как у автора.

## В продукте
- Режим: [[mode-X]]
- Упражнение: exercises.number = N
- Промпт: [[prompt-Y]]

## Связанные концепты
- [[другой-концепт]]
```

## Зачем это нужно

Граф с концептами показывает: если три книги говорят про "границы" разными словами — это повод для сводного режима (например, "чат про границы" по всем трём школам). Без графа это теряется.
