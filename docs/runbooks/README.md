# Runbooks

Пошаговые инструкции для типовых операций. Каждый runbook — самодостаточный, с copy-paste командами и описанием ожидаемого результата на каждом шаге.

## Индекс

| Runbook | Описание | Частота |
|---------|----------|---------|
| [add-new-book](add-new-book.md) | Добавление новой книги/программы | При каждой новой книге |
| [create-test](create-test.md) | Создание теста для программы | При каждом новом тесте |
| [prompt-tuning](prompt-tuning.md) | Тюнинг системных промптов | Регулярно |

## Как создать новый runbook

1. Скопируй `_TEMPLATE.md` → `operation-name.md`
2. Заполни все секции (особенно "Откат" и "Частые ошибки")
3. Добавь запись в таблицу выше
4. Коммит: `docs: add runbook for operation-name`
