## Структура заметки о книге

Рекомендуемая структура (пока не шаблон — просто ориентир):

```markdown
# Название книги — Автор

## Краткое содержание
Одним абзацем о главной идее.

## Архетипы
- [[archetype-X]] — описание
- [[archetype-Y]] — описание

## Режимы программы
- [[mode-A]] — использует [[prompt-A-v1]]
- [[mode-B]] — использует [[prompt-B-v1]]

## Связанные концепты
- [[concepts/boundaries|Границы]]
- [[concepts/self-soothing|Self-soothing]]

## Источники в БД
- programs.slug: `nice-guy` (или другой)
- program_modes: ссылка на миграцию
```
