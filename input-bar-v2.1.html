Idle: Микрофон в idle. Desktop (mouse): click = locked (hands-free). Mobile (touch): hold = push-to-talk, свайп ↑ = lock. Начни печатать → стрелка.
Кнопка 44→64px: При recording/locked кнопка вырастает до 64px (×1.45). align-items: center на input-bar — рост симметричный от центра, не ломает rec-bar.
Platform-aware: event.pointerType === 'touch' → hold = push-to-talk (отпустил = отправил), свайп ↑ = lock. pointerType === 'mouse' → click = сразу locked mode. Переключатель Auto/Touch/Mouse для тестирования.
Error bar: Показ 2.5с → плавный fadeOut 500ms → textarea возвращается. Dismiss кнопка убирает мгновенно. Не блокирует, не бесит.
Режимы: Chat/Exercise = mic + send + check. Test = send + check, без микрофона. Атрибут data-mode.
A11y + Dark: Все цвета через токены. [data-theme="dark"]. prefers-reduced-motion. aria-label динамический.
