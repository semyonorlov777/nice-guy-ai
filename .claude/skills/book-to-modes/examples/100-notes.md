# 100 записок о развитии — Ретроспектива

## Книга

- **Автор:** Пётр Осипов, сооснователь «Бизнес Молодость»
- **Издательство:** Эксмо, 2020, 464 стр., ISBN 978-5-04-115804-0
- **Slug:** `100-notes`
- **Тип:** Мотивация / мышление + Практика / навыки

## Особенность проекта

У нас не было полного текста книги — содержание реконструировано через Gemini Deep Research + открытые источники. Из 100 записок **18 подтверждены** из рецензий/цитат, остальные 82 реконструированы по тематическим кластерам.

## Ключевые решения

| # | Решение | Обоснование |
|---|---------|-------------|
| 1 | 7 уникальных режимов | 8 кластеров → 7 режимов. Кластеры 7+8 объединены (оба про рефлексию) |
| 2 | Prefix `notes_` | Короче чем `100_notes_`, нет конфликтов |
| 3 | Бизнес-лаборатория = Экзамен | Бизнес — ядро ЦА Осипова. Кейсовый формат = естественный экзамен |
| 4 | Дневник = продвинутый Анализ | Минимальный scaffolding, AI задаёт ОДИН вопрос |
| 5 | Деконструктор страхов = free | Страхи — входная точка для ЦА |
| 6 | Нет режима для тела/дыхания | Встроено в «Архитектор энергии» |
| 7 | Нет отдельного режима для денег | Пересечение «Масштаб мышления» + «Бизнес-лаборатория» |
| 8 | Братский, уязвимый тон | Книга = дневник, без гуру-позиции |

## Lessons learned

1. **Gemini Deep Research** — отличный инструмент для реконструкции без полного текста
2. **Мотивационные книги** сложнее структурировать, чем терапевтические (нет чёткой диагностики)
3. **Бизнес-тематика** требует осторожности с антипаттернами (не давать финансовых советов)
4. **Кросс-связки** проще когда режимы покрывают разные аспекты одной проблемы
