# Changelog

Все заметные изменения платформы Nice Guy AI.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [Unreleased]

### Добавлено
- Система документации проекта: ADR, runbooks, схема БД, справочник env
- Карточка теста на хабе программы Games People Play («Какие игры ты играешь?») — тест существовал в БД, но был невидим без mode_template/program_mode
- System prompt для GPP-теста (`programs.test_system_prompt`) — улучшает качество интерпретации текстовых ответов в тесте
- Quick-reply кнопки в режимах «Славный парень или нет?» и «Теория книги» (4 кнопки в каждом)
- Аудиты обеих программ с тестами (`.claude/skills/book-audit/findings/`)
- AI-приветствие на хабе программы Games People Play (3 варианта: первый визит / вернулся с тестом / вернулся без теста) — раньше на месте текста был пустой золотой кружок
- 5 тем для Games People Play с привязкой к шкалам теста (Твои игры, Родитель-Взрослый-Ребёнок, Жизненный сценарий, Голод по признанию, Треугольник Карпмана) — позволяют персонализировать хаб по результатам теста
- Дебаг-параметр `?test_state=welcome|history-single|history-multi` для визуального тестирования welcome-экрана теста без данных в БД

### Изменено
- `appendTestScores()` теперь сам инжектит правило «не зачитывай числовые баллы вслух» — раньше дублировалось в каждом промпте
- ta_diagnostic в Games People Play теперь корректно открывается с хаба (раньше вёл в свободный чат)
- Хаб больше не использует хардкод-таблицу tool keys — derive из mode.key, новый chat-based режим автоматически роутится без правок UI
- Удалён disabled режим `self_work` из nice-guy, sort_order перенумерован 0–9 без дублей
- Чеклист `book-audit` v1.5: добавлены проверки `programs.hub_messages` (E13) и динамического заголовка `HistoryScreen` (E14); новая секция I про паттерн `?<screen>_state=` для дебага UI-состояний

### Исправлено
- Экран истории теста больше не показывает «Индекс Синдрома Славного Парня» для программ, отличных от nice-guy — заголовок и бейдж берутся из `testConfig.ui_config.welcome_title/welcome_badge`

## [2026-03-28]

### Добавлено
- Google OAuth — вход через Google аккаунт
- Тест для "Игры, в которые играют люди" — GPP-тест с 7 шкалами
- Универсальная тест-система — тесты для любой книги через конфигурацию в БД
- Тематические чаты: Границы, Родители, Отношения, Синдром
- Квиз-режим для программ
- Теория-режим для программ

### Изменено
- Тест-система переведена на generic slug-based routing (вместо хардкода ISSP)
- TestCardFlow рефакторинг — выделены хуки useTestInit, useTestSession, useTestAnswers, useResultPolling
