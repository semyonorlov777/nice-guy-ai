# Changelog

Все заметные изменения платформы Nice Guy AI.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [2026-03-28]

### Добавлено
- Google OAuth — вход через Google аккаунт
- Тест для "Игры, в которые играют люди" — GPP-тест с 7 шкалами
- Универсальная тест-система — тесты для любой книги через конфигурацию в БД
- Тематические чаты: Границы, Родители, Отношения, Синдром
- Квиз-режим для программ
- Теория-режим для программ

### Изменено
- Тест-система переведена на generic slug-based routing (вместо хардкода ISSP)
- TestCardFlow рефакторинг — выделены хуки useTestInit, useTestSession, useTestAnswers, useResultPolling

## [2026-03-21]

### Добавлено
- Программа "Игры, в которые играют люди" (Эрик Берн)
- Программа "5 языков любви" (Гэри Чепмен)
- Хаб программы — единый экран выбора режимов
- Система режимов (mode_templates + program_modes) — data-driven UI

### Изменено
- Главная страница — каталог программ вместо одной книги
- Sidebar и MobileTabs адаптированы под мультипрограммность

## [2026-03-14]

### Добавлено
- Голосовой ввод — запись голоса + транскрипция через API
- Яндекс OAuth — вход через Яндекс аккаунт
- Портрет пользователя — AI-анализ на основе переписки

### Изменено
- AuthSheet — единый компонент авторизации (sheet/fullscreen)
- Тройное обнаружение авторизации (postMessage + onAuthStateChange + polling)

## [2026-03-07]

### Добавлено
- Тест ISSP — 35 вопросов, 7 шкал, radar-диаграмма
- Анонимные тесты — прохождение без регистрации
- Auth wall на Q34 — мягкая конверсия в авторизацию
- AI-интерпретация результатов теста
